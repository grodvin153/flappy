3 - Funcionamiento de juego

Movimiento ascendente del pájaro

Por último, vamos a añadir el movimiento ascendente del pájaro. Este movimiento se producirá con la pulsación de cualquier tecla del teclado.

Para ello, en primer lugar vamos a definir el valor de vuelo que tendrá el pájaro con la variable FLAP_VELOCITY. Esta variable tendrá valor -6.5.

El valor negativo, supondrá una ascensión en el movimiento vertical.

ver código

FLAP_VELOCITY = -6.5

A continuación definimos la función on_key_down. Esta función está reservada en Pygame para cualquier pulsación de teclado.

Dentro de la función se asigna el valor de la velocidad vertical vy al valor de la variable de aleteo FLAP_VELOCITY.

No hace falta cambiar la posición vertical del pájaro, ya que esto se realizará al llamar a la función update_bird, algo que se produce de forma constante.

ver código

def on_key_down():
    bird.vy = FLAP_VELOCITY

Vamos a hacer una pequeña condición al código anterior: el pájaro sólo vuela si está vivo. De esta forma controlamos el correcto funcionamiento del juego cuando ocurran colisiones.

La condición a añadir sería si (if) no (not) está muerto (dead) el actor pájaro (bird).

ver código

Cambiamos el código

def on_key_down():
    bird.vy = FLAP_VELOCITY

Por este otro

def on_key_down():
    if not bird.dead:
        bird.vy = FLAP_VELOCITY

Cuidado al añadir una nueva condición, ya que el código que va dentro de ella tiene que desplazarse a la derecha.

Colisión de pájaro con tubería

A continuación vamos a controlar la colisión entre el pájaro y las tuberías, aprovechando el uso del atributo dead que tiene el actor bird.

  • Dentro de la función update_bird:
    • añadimos la condición si (if) el pájaro está colisionando (colliderect) con la tubería superior (pipe_top) o (or) colisiona con la tubería inferior (pipe_bottom)
      • En caso que colisione, el valor de muerto (dead) se cambia a true.
      • Cambiamos el aspecto del pájaro a la imagen birddead.

ver código

Inicialmente tenemos la función update_bird

def update_bird():
    bird.vy += GRAVITY
    bird.y += bird.vy
    bird.x = 75

Añadimos el siguiente código:

    if bird.colliderect(pipe_top) or bird.colliderect(pipe_bottom):
        bird.dead = True
        bird.image = 'birddead'

Una vez el pájaro ha muerto, podemos reinicializar el juego, recuperando la posición inicial del pájaro, su imagen inicial y la reiniciando las posiciones de las tuberías.

Para ello se añadirá el siguiente código:

  • Una nueva condición que controle si la posición vertical del pájaro y está fuera de la ventana, esto es no está entre 0 y 720.
    • se reinicia la posición vertical del pájaro y a 200.
    • se "revive" al pájaro cambiando el valor dead a true.
    • se reinicia la velocidad vertical vy a 0, aunque inmediatamente sufrirá una nueva aceleración a causa de la gravedad.
    • se reinicia la posición de las tuberías llamando a la función reset_pipe.

ver código

Tenemos la función update_bird con este código:

def update_bird():
    uy = bird.vy
    bird.vy += GRAVITY
    bird.y += bird.vy
    bird.x = 75
    if bird.colliderect(pipe_top) or bird.colliderect(pipe_bottom):
        bird.dead = True
        bird.image = 'birddead'

Y añadimos esta nueva condición

    if not 0 < bird.y < 720:
        bird.y = 200
        bird.dead = False
        bird.vy = 0
        reset_pipes()

Operadores lógicos and, or, not

Al utilizar condiciones podemos aprovechar los operadores lógicos para construir sentencias más complejas y que abarquen condiciones más precisas o más amplias. Para ello contamos con los siguientes operadores:

  • And (&): devuelve true si todas las condiciones son ciertas
if (x > 5) and  (x < 10):  # sólo será válido para valores de x mayores a 5 y menores a 10
    • Or (|): devuelve true si al menos una de las condiciones es cierta
if (x < 5) and  (x > 10):  # será válido para cualquier valor de x menor a 5 y también cualquier valor de x mayor a 10
    • Not (!): devuelve true si la condición no es cierta
if not (x > 5):  # será válido para valores de x menores o iguales a 5

Efecto de movimiento del pájaro

Para incluir un efecto de movimiento en el pájaro, cambiaremos su imagen según esté subiendo (vy<-1) o cayendo (vy>=0).

  • Creamos una condición que compruebe que el pájaro no está muerto (dead). Así evitamos cambiar la imagen ya asignada de pájaro muerto.
    • Creamos una nueva condición (if), en la que si la velocidad vertical vy es menor que -1 (está subiendo)
      • el pájaro tenga la imagen bird2.
    • En caso contrario (else), es decir la velocidad vertical es mayor o igual a 0 (está quieto o cayendo)
      • el pájaro tenga la imagen inicial bird1

ver código

Cambiamos el código de la función update_bird

def update_bird():
    bird.vy += GRAVITY
    bird.y += bird.vy
    bird.x = 75
    if bird.colliderect(pipe_top) or bird.colliderect(pipe_bottom):
        bird.dead = True
        bird.image = 'birddead'
    if not 0 < bird.y < 720:
        bird.y = 200
        bird.dead = False
        bird.vy = 0
        reset_pipes()

Añadiendo la siguiente condición

    if not bird.dead:
        if bird.vy < -1:
            bird.image = 'bird2'
        else:
            bird.image = 'bird1'

Condicionales anidados

Los condicionales, permiten escribir código en su interior y en realidad, nada de impide incluso al interior de un condicional, poner otros (u otros). A eso se le llama condiciones anidados, pues una estructura condicional dentro de otra. De hecho, puedes anidar cuantos condicionales requieras, aunque no se recomienda más de dos o tres niveles.

if z > 8:
  print("Es mayor que 8") 
else:
  if z > 6:
      print("Es mayor que 6 pero menor o igual que 8") 
  else:
      if z > 5:
          print("Es mayor que 5 pero menor o igual que 6") 
      else:
          print("Es menor o igual que 5")

Dentro de un condicional, puedes poner cualquier instrucción válida y eso incluye a cualquier tipo de condicional que necesites y el funcionamiento, sigue siendo el mismo. De hecho, Python tiene una instrucción llamada elif() que permite simplificar un if() que se encuentra al interior de un else en Python.

if z > 8:
  print("Es mayor que 8") 
elif z > 6:
  print("Es mayor que 6 pero menor o igual que 8") 
elif z > 5:
  print("Es mayor que 5 pero menor o igual que 6") 
else:
  print("Es menor o igual que 5")
